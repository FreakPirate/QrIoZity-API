Category

You can add Categories to our API. It is as simple as that. Just click the button below, or at the navigation bar, it will lead you to the "Category Append" page. Fill the category details. And thats it.

Try to avoid entering redundant categories. You could always see the list of already available categories down below in the page. So first consult the list and then fill up the category details.

API Usage (Documentation)

/getCategories

  • Sample Url:http://bets.zapto.org/getcategories
  • Description: It will fetch the array of available categories in JSON format.
  • Example:
      [{"id":4,"categ_name":"aptitude"},{"id":2,"categ_name":"electronics"},{"id":3,"categ_name":"science"},{"id":1,"categ_name":"test"}]

/quizQuestionsByCategory/:categoryId

  • Sample Url:http://bets.zapto.org/quizQuestionsByCategory/2
  • Description: It will fetch Questions from a Category as given by categoryId.
  • Example:
      [{"id":1,"trivia_categories_id":3,"question":"Which of the following best describes Uranium ?","choices":"Heavy Metal|Organic Compund|Radioactive|Inert Gas","answer":"3","tags":"Science, Math, Knowledge","date_created":"2015-03-22T16:29:31.000Z"}]

/allQuizQuestions

  • Url:http://bets.zapto.org/allQuizQuestions
  • Description: It will fetch the data relating to all the questions stored in the database in the form of a JSON String.
  • Options (This option is optional)
    • page(int): Specifying the page number to fetch as JSON Array.
      • Url: http://bets.zapto.org/allQuizQuestions?page=1
      • Example:
      • [{"id":11,"trivia_categories_id":4,"question":"The probability that 'A' speaks truth is 4/5, while this probability for 'B' is 3/4. The probability that they contradict each other when asked to speak on a fact is:","choices":"0.35|0.25|0.30|0.25","answer":"1","tags":"","date_created":"2015-03-25T16:25:45.000Z"}]
    • limit(int): It limits the number of questions per page.
      • Url: http://bets.zapto.org/allQuizQuestions?page=1&limit=4
      • Example:
      • [{"id":1,"trivia_categories_id":3,"question":"Which of the following best describes Uranium ?","choices":"Heavy Metal|Organic Compund|Radioactive|Inert Gas","answer":"3","tags":"Science, Math, Knowledge","date_created":"2015-03-22T16:29:31.000Z"},{"id":2,"trivia_categories_id":4,"question":"In a given test by prof. virus raju gets 32% marks and fails by 20 marks. If the century under consideration is 20th century, what is the % of people born on 29th feb ?\r\n[assume no deaths]","choices":"0.685|0.0685|0.684|0.0684","answer":"4","tags":"","date_created":"2015-03-22T17:05:46.000Z"},{"id":4,"trivia_categories_id":4,"question":"There were 'p' pigeons and 's' sparrows in a cage.
